El elemento p de párrafo es el apropiado para distribuir el texto en párrafos, esto es una etiqueta p.

El elemento header representa un contenedor para contenido introductorio o un conjunto de enlaces de navegación. Lo rojo esta dentro de una etiqueta header.

El elemento em es el apropiado para marcar con énfasis las partes importantes de un texto.

El elemento strong es el apropiado para marcar con especial énfasis las partes más importantes de un texto.

Ejemplo:

La etiqueta em es importante pero la etiqueta strong es más importante.

El elemento aside representa una sección de una página que consiste en contenido que está indirectamente relacionado con el contenido principal del documento, lo color gris esta dentro de una etiqueta aside.

La etiqueta b es para negritas

El elemento div es el contenedor genérico para el contenido de flujo. No tiene ningún efecto sobre el contenido o el diseño hasta que se le aplica algún tipo de estilo mediante CSS (por ejemplo, se le aplica directamente un estilo o se aplica algún tipo de modelo de diseño como Flexbox a su elemento padre). lo coral esta dentro de una etiqueta div

La etiqueta i pondra el texto en italicas

La etiqueta u subraya el texto

El elemento section representa una sección independiente genérica de un documento, que no tiene un elemento semántico más específico para representarla. Las secciones siempre deben tener un encabezamiento, con muy pocas excepciones. Lo color amarillo esta dentro de una etiqueta section

Esto es una etiqueta del que tachara el contenido y esto una etiqueta ins que remarcara el correcto!

El elemento article representa una composición autocontenida en un documento, página, aplicación o sitio, destinada a ser distribuida o reutilizada de forma independiente (por ejemplo, en sindicación). Algunos ejemplos son: una entrada de foro, un artículo de revista o periódico, o una entrada de blog, una ficha de producto, un comentario enviado por un usuario, un widget o gadget interactivo, o cualquier otro elemento independiente de contenido. Lo color rosa esta dentro de una etiqueta article

El elemento nav representa una sección de una página cuyo propósito es proporcionar enlaces de navegación, ya sea dentro del documento actual o a otros documentos. Ejemplos comunes de secciones de navegación son menús, tablas de contenido e índices. Estoy dentro de una etiqueta nav (azul clarito).
